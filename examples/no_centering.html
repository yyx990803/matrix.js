Disabling auto centering

By default, Matrix.js automatically center objects with the following transform: 

translate3d(-50%, -50%, 0)
And every MX.Object3d comes with the class .mx-object3d. So you can auto center all objects inside a container with the following CSS rules: 
.mx-object3d {
    position: absolute;
    top: 50%;
    left: 50%;
}

But sometimes we simply want to add 3D movement to exsiting DOM nodes without tampering with their positioning. This page shows that you can disable the default centering behaviour by setting:

MX.positionAtCenter = false
